# 惡魔果實 2D — Blox Fruits 2D

> **作者：Sam Lin**  
> **授權：MIT License（免費、可分享、可修改）**  
> **版本：v1.0 MVP (2026)**

一款使用 Python + Pygame 打造的 2D 頂視角動作 RPG，靈感來自惡魔果實冒險題材。  
低配備、快節奏、高爽感 — 專為課後快樂遊玩設計。

---

## 快速開始

### 需求
- Python 3.8 以上 ([下載](https://www.python.org/downloads/))
- Windows 7/10/11

### 安裝與執行（一鍵）

```
1. 雙擊 setup_venv.bat    ← 建立虛擬環境並安裝 pygame
2. 雙擊 run_game.bat      ← 啟動遊戲
```

### 手動執行（進階）

```bash
python -m venv venv
venv\Scripts\pip install pygame numpy
venv\Scripts\python main.py
```

---

## 操作說明

| 按鍵 | 動作 |
|------|------|
| `WASD` 或 方向鍵 | 移動角色 |
| `J` | 基礎攻擊（發射子彈） |
| `K` | 果實技能（需裝備果實） |
| `T` | 開啟果實祭壇（需靠近金色六邊形） |
| `~` 或 `\` | 輸入密技碼 |
| `ESC` | 暫停選單 |
| `R` (死亡後) | 重新開始 |

## 遊戲機制

### 核心循環
```
擊殺敵人 → 獲得碎片 → 前往祭壇 → 覺醒果實 → 變強 → 擊殺 Boss
```

### 果實系統

| 果實 | 稀有度 | 被動效果 | 技能 |
|------|--------|----------|------|
| 橡膠果實 | 普通 (70%) | +20% 迴避率 | 橡膠子彈亂打（5連發） |
| 火焰果實 | 稀有 (25%) | +15% 全體傷害 | 火焰大地獄（範圍爆炸） |
| 冰凍果實 | 稀有 (20%) | 攻擊附帶減速 | 冰河時代（凍結 AOE） |
| 龍龍果實 | 傳說 (5%) | +50% 攻擊力 | 龍息吐息（穿透光線） |
| 豹豹果實 | 傳說 (5%) | +40% 暴擊率 | 獸王獠牙（多段衝刺） |

> 收集 **10 片**同種碎片 → 前往祭壇（地圖中央金色圖案）→ 按 `T` 覺醒！

### Boss 出現條件
- 擊殺 **20 隻**敵人 → 赤犬登場
- 擊殺 **40 隻**敵人 → 黃猿登場
- 擊殺 **60 隻**敵人 → 青雉登場

### 等級里程碑

| 等級 | 里程碑 | 獎勵 |
|------|--------|------|
| Lv.10 | 新手畢業 | +1000 EXP +200 金幣 |
| Lv.20 | 進階戰士 | +3000 EXP +免費抽卡 |
| Lv.30 | 強者降臨 | +8000 EXP +3次抽卡 |
| Lv.50 | 傳奇開始 | +20000 EXP +5次抽卡 |
| Lv.99 | 終極之力 | 終極獎勵 |

## 密技碼（彩蛋）

在遊戲中按 `~` 輸入以下密技碼：

| 密技碼 | 效果 | 次數限制 |
|--------|------|----------|
| `GODMODE` | 無敵 30 秒 | 1次 |
| `MONEYRAIN` | 獲得 10,000 金幣 | 3次 |
| `GACHA10` | 免費抽取 10 次果實碎片 | 1次 |
| `LEVELUP` | 獲得 50,000 EXP | 2次 |
| `ALLFRUITS` | 解鎖所有果實碎片 | 1次 |
| `MAXLEVEL` | 直升 50 級 | 1次 |
| `KILLAURA` | 消滅畫面所有敵人 | 3次 |

## 技術架構

```
BloxFruits/
├── main.py                  # 遊戲入口 (60 FPS 主迴圈)
├── config.py                # 全域設定
├── core/                    # 核心引擎 (常數、事件系統)
├── entities/                # 遊戲實體 (玩家、敵人、Boss、子彈、道具)
├── managers/                # 管理器 (狀態機、鏡頭、碰撞、存檔、密技碼)
├── systems/                 # 系統 (傷害計算、升級、扭蛋、粒子、音效)
├── ai/                      # 敵人 AI (巡邏、追蹤、Boss 行為樹)
├── ui/                      # UI (HUD、選單、暫停、祭壇、浮動文字)
├── content/                 # JSON 資料表 (敵人、Boss、果實、密技碼)
└── utils/                   # 工具函式 (資料載入、數學計算)
```

**技術棧：** Python 3.10+ | Pygame 2.6 | NumPy (音效生成)  
**無需任何外部美術資源** — 所有畫面均以 Pygame 程式繪製

# 惡魔果實 2D — Blox Fruits 2D

> **作者：Sam Lin**  
> **授權：MIT License（免費、可分享、可修改）**  
> **版本：v1.1.0 (2026)**

一款使用 Python + Pygame 打造的 2D 頂視角動作 RPG，靈感來自惡魔果實冒險題材。  
低配備、快節奏、高爽感 — 專為課後快樂遊玩設計。

## 快速開始

### 需求

- Python 3.8 以上（[下載](https://www.python.org/downloads/)）
- Windows 7/10/11

### 安裝與執行（一鍵）

```
1. 雙擊 setup_venv.bat    ← 建立虛擬環境並安裝 pygame
2. 雙擊 run_game.bat      ← 啟動遊戲
```

### 手動執行（進階）

```bash
python -m venv venv
venv\Scripts\pip install pygame numpy
venv\Scripts\python main.py
```

## 操作說明

| 按鍵             | 動作                             |
| ---------------- | -------------------------------- |
| `WASD` 或 方向鍵 | 移動角色（頭上箭頭顯示朝向）     |
| `J`              | 基礎攻擊（發射子彈）             |
| `K`              | 果實技能（需裝備果實）           |
| `H`              | 使用生命藥水（需在商店購買儲存） |
| `T`              | 開啟果實祭壇（需靠近金色六邊形） |
| `~` 或 `\`       | 輸入密技碼                       |
| `ESC`            | 暫停選單                         |
| `R` (死亡後)     | 重新開始                         |

## 遊戲機制

### 核心循環

```
擊殺敵人 → 獲得碎片/金幣 → 前往祭壇 → 覺醒果實/購買道具/抽卡 → 變強 → 擊殺 Boss → 進入下一關卡
```

### 果實系統

| 果實     | 稀有度     | 被動效果      | 技能                   | 碎片需求 |
| -------- | ---------- | ------------- | ---------------------- | -------- |
| 橡膠果實 | 普通 (56%) | 攻擊免傷 30%  | 橡膠子彈亂打（5連發）  | 10 片    |
| 火焰果實 | 稀有 (20%) | +15% 全體傷害 | 火焰大地獄（範圍爆炸） | 10 片    |
| 冰凍果實 | 稀有 (16%) | 攻擊附帶減速  | 冰河時代（凍結 AOE）   | 10 片    |
| 龍龍果實 | 傳說 (4%)  | +50% 攻擊力   | 龍息吐息（穿透光線）   | 10 片    |
| 豹豹果實 | 傳說 (4%)  | +40% 暴擊率   | 獸王獠牙（多段衝刺）   | 10 片    |

> 收集 **10 片**同種碎片 → 前往祭壇（地圖中央金色圖案）→ 按 `T` 覺醒！

### 祭壇系統（按 T 開啟）

| 頁籤       | 功能                             |
| ---------- | -------------------------------- |
| ⚔ 果實覺醒 | 消耗 10 片碎片覺醒對應果實       |
| 🛒 商店    | 用金幣購買強化道具與藥水         |
| 🎰 抽卡    | 花費金幣或抽卡券獲得隨機果實碎片 |

### 商店道具說明

| 道具     | 費用  | 效果                                 |
| -------- | ----- | ------------------------------------ |
| 碎片扭蛋 | 100 G | 隨機獲得 1 個惡魔果實碎片            |
| 生命藥水 | 200 G | 加入藥水背包（最多 5 瓶），按 H 使用 |
| 力量強化 | 500 G | 本局永久 +8 攻擊力（即時生效）       |
| 防禦強化 | 100 G | 本局永久 +8 防禦力（即時生效）       |
| 幸運符   | 350 G | 本局永久 +10% 暴擊率（即時生效）     |

### 抽卡系統

| 抽法  | 費用              | 說明                   |
| ----- | ----------------- | ---------------------- |
| 1 抽  | 100 G 或 1 張票   | 獲得 1 個隨機果實碎片  |
| 10 抽 | 1000 G 或 10 張票 | 獲得 10 個隨機果實碎片 |

- 抽卡券可從**等級里程碑**免費獲得（Lv.20、Lv.30、Lv.50）
- 10 碎片同種果實即可前往覺醒頁籤覺醒

### Boss 出現條件

每個區域擊殺 **20 隻**敵人後，區域 Boss 登場！

| 順序 | Boss 名稱 | 特色              | 所在區域 |
| ---- | --------- | ----------------- | -------- |
| 第 1 | 赤犬      | 岩漿拳 + 大帽子   | 新手海島 |
| 第 2 | 黃猿      | 金色光芒 + 大墨鏡 | 火山基地 |
| 第 3 | 青雉      | 冰柱 + 懶散表情   | 傳說城堡 |

> Boss 死亡後，玩家自動傳送至下一個區域中央！

### 等級里程碑

| 等級  | 里程碑   | 獎勵                           |
| ----- | -------- | ------------------------------ |
| Lv.10 | 新手畢業 | +1000 EXP、+200 金幣           |
| Lv.20 | 進階戰士 | +3000 EXP、+500 金幣、1 抽券   |
| Lv.30 | 強者降臨 | +8000 EXP、+1000 金幣、3 抽券  |
| Lv.50 | 傳奇開始 | +20000 EXP、+3000 金幣、5 抽券 |
| Lv.99 | 終極之力 | 終極獎勵                       |

## 密技碼（彩蛋）

在遊戲中按 `~` 輸入以下密技碼：

| 密技碼      | 效果                   |
| ----------- | ---------------------- |
| `GODMODE`   | 無敵 30 秒             |
| `MONEYRAIN` | 獲得 10,000 金幣       |
| `GACHA10`   | 免費抽取 10 次果實碎片 |
| `LEVELUP`   | 獲得 50,000 EXP        |
| `ALLFRUITS` | 解鎖所有果實碎片       |
| `MAXLEVEL`  | 直升 50 級             |
| `KILLAURA`  | 消滅畫面所有敵人       |

## 技術架構

```
game-blox-fruits-2d/
├── main.py                  # 遊戲入口 (60 FPS 主迴圈)
├── config.py                # 全域設定 + 版本號
├── core/                    # 核心引擎 (常數、事件系統)
├── entities/                # 遊戲實體 (玩家、敵人、Boss、子彈、道具)
├── managers/                # 管理器 (狀態機、鏡頭、碰撞、存檔、密技碼)
├── systems/                 # 系統 (傷害計算、升級、扭蛋、粒子、音效)
├── ai/                      # 敵人 AI (巡邏、追蹤、Boss 行為樹)
├── ui/                      # UI (HUD、選單、暫停、祭壇、浮動文字)
├── content/                 # JSON 資料表 (敵人、Boss、果實、密技碼)
├── release/                 # 打包工具 (spec、build.bat、icon 生成)
└── utils/                   # 工具函式 (資料載入、數學計算)
```

**技術棧：** Python 3.10+ | Pygame 2.6 | NumPy（音效生成）  
**無需任何外部美術資源** — 所有畫面均以 Pygame 程式繪製

## 存檔系統

- 遊戲存檔於 `save_data.json`（與 main.py 同目錄）
- **自動存檔**：擊敗 Boss 進入下一區域時自動儲存
- **自動存檔**：按 ESC → 返回主選單時自動儲存
- **手動存檔**：暫停選單（ESC）→「💾 儲存遊戲」
- 進度包含：等級、金幣、果實、碎片、擊殺數、藥水庫存、抽卡券
- HUD 底部提示每 10 秒交替顯示操作說明與存檔提醒
